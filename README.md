## üîç Key Steps Performed
1. **Data Import & Cleaning**
   - Removed duplicates and missing values.
   - Formatted date and numerical columns.
   - Added calculated fields for analysis.

2. **Exploratory Data Analysis (EDA)**
   - Used pivot tables to find cancellation patterns.
   - Analyzed relationships between cancellation rates and:
     - Lead time
     - Deposit type
     - Market segment
     - Customer type
     - Room type

3. **Visualization**
   - Created bar charts and pie charts to display insights clearly.

## üìà Insights & Findings
- Higher cancellation rates were found in **City Hotels** compared to **Resort Hotels**.
- Bookings with **no deposit** showed a higher tendency to cancel.
- Longer **lead times** were associated with increased cancellations.
- **Online travel agencies (OTA)** had a significant share of cancellations compared to direct bookings.
